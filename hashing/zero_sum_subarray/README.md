Given an array of integers A, find and return whether the given array contains a non-empty subarray with a sum equal to 0.

If the given array contains a sub-array with sum zero return 1, else return 0.  

Example 1:
```buildoutcfg
Input:
    A = [1,2,3,4,5]
Output:
    result = 0
```

Example 2:
```buildoutcfg
Input:
    A = [-1, 1]
Output:
    result = 1
```
